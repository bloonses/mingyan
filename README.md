# 鸣言 (Mingyan)

一个基于 Python 的全中文语法编程语言。

## 简介

鸣言是一门使用中文关键字的现代编程语言，由词法分析器、语法解析器和树遍历解释器构成。它支持变量、函数、类、异常处理、闭包、列表推导式等现代编程语言特性，所有语法关键字均采用中文表达，降低了中文使用者的编程入门门槛。

## 特性

- **全中文关键字** — 变量、函数、类、控制流等语法全部使用中文
- **完整数据类型** — 整数、浮点数、字符串、布尔、空值、列表、字典
- **控制流** — `如果`/`否则如果`/`否则`、`当`、`遍历`...`在`
- **函数与闭包** — 支持递归、默认参数、闭包、匿名函数
- **面向对象** — `类`、`继承`、`新`、`这个`，支持方法重写
- **异常处理** — `尝试`/`捕获`/`最终`/`抛出`
- **高级表达式** — 三元表达式、列表推导式、切片访问
- **模块系统** — `导入`/`从`...`导入`
- **丰富内置函数** — 数学、文件 I/O、类型检查、列表/字符串/字典方法
- **交互式 REPL** — 支持多行输入、帮助信息
- **可视化编辑器** — 附带基于 Web 的鸣言编辑器，支持一键启动

## 快速开始

### 环境要求

- Python 3.6+
- Node.js（使用编辑器时需要）

### 启动方式

```bash
# 启动交互式环境 (REPL)
python src/huayen.py

# 执行鸣言源文件
python src/huayen.py examples/示例.鸣

# 直接执行一行代码
python src/huayen.py -e "打印(\"你好，世界！\")"

# 显示帮助信息
python src/huayen.py -h
```

### 鸣言编辑器

项目附带了一个基于 Web 的可视化编辑器，打包在 `editor.7z` 中。

**安装步骤：**

1. 解压 `editor.7z` 到项目目录，得到 `editor` 文件夹
2. 确保已安装 [Node.js](https://nodejs.org/)

**启动方式：**

- **一键启动（Windows）**：解压 `editor.7z` 后，进入 `editor` 文件夹，双击 `启动编辑器.bat`，脚本会自动安装依赖并启动开发服务器，随后在浏览器中打开编辑器
- **手动启动**：
  ```bash
  cd editor
  npm install
  npm run dev
  ```
  然后在浏览器中访问 `http://localhost:5173/`

## 语法概览

### 变量与常量

```
变量 名字 = "鸣言"
常量 PI = 3.14
名字 = "新名字"
```

### 数据类型

```
变量 整数 = 42
变量 浮点 = 3.14
变量 字符串 = "你好"
变量 布尔值 = 真
变量 空值 = 空
变量 列表 = [1, 2, 3]
变量 字典 = {"名字": "鸣言", "版本": 1.0}
```

### 运算符

| 类别 | 运算符 |
|------|--------|
| 算术 | `+` `-` `*` `/` `%` `**` |
| 比较 | `==` `!=` `<` `>` `<=` `>=` |
| 逻辑 | `且` `或` `非` |
| 赋值 | `=` `+=` `-=` `*=` `/=` |

### 条件判断

```
变量 分数 = 85
如果 分数 >= 90 {
    打印("优秀！")
} 否则如果 分数 >= 80 {
    打印("良好！")
} 否则 {
    打印("及格")
}
```

### 循环

```
遍历 i 在 范围(1, 11) {
    打印(i)
}

变量 倒计时 = 5
当 倒计时 > 0 {
    打印(倒计时)
    倒计时 -= 1
}
```

### 函数

```
函数 阶乘(n) {
    如果 n <= 1 {
        返回 1
    }
    返回 n * 阶乘(n - 1)
}
打印(阶乘(5))
```

### 类与继承

```
类 动物 {
    函数 初始化(名字) {
        这个.名字 = 名字
    }
    函数 叫声() {
        打印("...")
    }
}

类 猫 继承 动物 {
    函数 叫声() {
        打印("喵喵喵~")
    }
}

变量 小猫 = 新 猫("小花")
小猫.叫声()
```

### 异常处理

```
尝试 {
    变量 结果 = 10 / 0
} 捕获 错误 {
    打印("捕获到错误: " + 错误)
} 最终 {
    打印("执行完毕")
}
```

### 高阶函数

```
变量 数字 = [1, 2, 3, 4, 5]
变量 偶数 = 过滤(函数(x) { 返回 x % 2 == 0 }, 数字)
变量 平方 = 映射(函数(x) { 返回 x * x }, 数字)
变量 总和 = 归约(函数(a, b) { 返回 a + b }, 数字, 0)
```

## 内置函数

| 函数 | 说明 |
|------|------|
| `打印(值)` | 输出值 |
| `长度(容器)` | 获取长度 |
| `类型(值)` | 获取类型名 |
| `输入(提示)` | 读取用户输入 |
| `转整数(值)` / `转浮点(值)` / `转字符串(值)` | 类型转换 |
| `范围(起始, 结束)` | 生成整数列表 |
| `排序(列表)` / `反转(列表)` | 排序与反转 |
| `映射(函数, 列表)` / `过滤(函数, 列表)` / `归约(函数, 列表)` | 函数式操作 |
| `平方根(值)` / `幂(底, 指)` / `绝对值(值)` | 数学函数 |
| `随机整数(a, b)` / `随机浮点()` | 随机数 |
| `文件读取(路径)` / `文件写入(路径, 内容)` / `文件存在(路径)` | 文件操作 |
| `转JSON(值)` / `从JSON(字符串)` | JSON 处理 |
| `是数字(值)` / `是字符串(值)` / `是列表(值)` | 类型检查 |
